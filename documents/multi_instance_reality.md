# 多实例 Reality 使用说明

## 这是什么

`19.多实例Reality` 用来部署多个**独立**的 Reality 节点。

它和 `12.添加新端口` 不一样：

* `12.添加新端口`：只是给旧节点补一个入口端口，后面还是同一个节点
* `19.多实例Reality`：会生成新的独立节点配置、新端口、新用户数据

## 当前范围

当前版本先支持：

* 当前核心为 `Xray-core` 时，部署多个独立的 `VLESS+Reality+Vision` 节点
* 当前核心为 `sing-box` 时，部署多个独立的 `VLESS+Reality+Vision` 节点

当前不做：

* 双核心同时托管
* 全协议多实例
* 把多实例节点混进旧的 `7.账号/订阅管理`

## 使用入口

安装完成后执行：

```bash
vasma
```

然后进入：

```text
19.多实例Reality
```

可用操作：

* 部署新的独立节点
* 查看已部署节点
* 查看某个节点账号
* 给某个节点添加用户
* 删除某个节点用户
* 删除某个节点

## 设计原则

* 多实例节点只作用于“当前已安装核心”
* 不去改写旧的单实例主流程
* 优先保证稳定性，而不是一次把所有协议全做成多实例
